# Contribution Guidelines

## What belongs here

- A public platform, dataset, or project that helps people in the Philippines engage with government, public information, or their communities.
- A direct Philippine connection and a working link.
- A description that names the user and the task the project supports.

## What does not belong

- Generic government software with no Philippine use case.
- Party campaigns, candidate promotion, or unverified political claims.
- Landing pages with no usable product.
- AI-generated link dumps.

## Pull request checklist

- [ ] Link works.
- [ ] Description explains what the project does and who can use it.
- [ ] Entry is in the right section and alphabetical order.
- [ ] No duplicate already exists.

## Description style

Bad:

> A revolutionary platform for government transformation.

Good:

> Portal for filing and browsing Freedom of Information requests to participating agencies.
